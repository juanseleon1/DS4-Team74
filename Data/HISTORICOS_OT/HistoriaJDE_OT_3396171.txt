BAJO ESTA OT SE REALIZA LA REVISION DEL SISTEMA DE PLANTAS DIESEL DE LA SUBESTACION DE 44.Y SE REALIZA EL SIGUIENTE INFORME.

INFORME TRABAJO REALIZADO EN EL SISTEMA LLENADO DE ACPM EN PLANTAS DIÉSEL PORCE 3.

EN LAS PLANTAS DIÉSEL SE REVISÓ EL CONTROL DE LLENADO DE LAS PLANTAS DIÉSEL ENCONTRÁNDOSE VARIOS INCONVENIENTES:
1. SISTEMA NO TRABAJA EN AUTOMÁTICO 
2. SISTEMA NO INDICA EL NIVEL DEL TANQUE PRINCIPAL 
3. SISTEMA FUNCIONA EN MANUAL, PERO NO ACTÚA EL CONTROL NIVEL DE LAS PLANTAS DIÉSEL 
4. PLANTA DIÉSEL NUMERO 1 NO TRABAJA BOMBA.


SOLUCIÓN. 
1. SE ENCUENTRA QUE EL SISTEMA EN AUTOMÁTICO DEPENDE DEL SENSOR DE NIVEL DEL TANQUE PRINCIPAL. 
LA SEÑAL DE DICHO SENSOR ESTÁ LLEGANDO A AL INDICADOR MAXTHERMO MC 5438. ESTE NO ES EL INDICADO YA QUE NO TIENE ENTRADA DE CORRIENTE, ESTO HIZO QUE SE ALARMARA Y ACTIVARÁ EL CONTACTO DE NIVEL BAJO DE ACPM INHABILITANDO LAS BOMBAS SUMERGIBLES DE LAS PLANTAS DIÉSEL. 
SE REEMPLAZA POR EL TRANSDUCTOR PR4501.  ESTE SE ESCALIZA DE 0.7M A 2.30M. RANGOS DEL SENSOR DE NIVEL Y SE INSTALA AL INTERIOR DEL GABINETE. 
A DEMÁS DE ESTO DE LE CONFIGURA LA ALARMA POR BAJO NIVEL EN 0.9M. 
CON EL CAMBIO DEL TRANSDUCTOR SE EVIDENCIARON ALGUNOS CAMBIOS QUE SE LE HABÍA HECHO AL CONTROL PARA QUE TRABAJARA EN MANUAL. ESA ERA LA RAZÓN POR LA CUAL EL SISTEMA NO TOMABA EN CUENTA EL NIVEL DE LAS PLANTAS DIÉSEL PARA PARAR LAS BOMBAS.  
SE NORMALIZÓ EL CIRCUITO DE CONTROL DEJANDO LAS PROTECCIONES DE NIVEL HABILITADAS.  


2. CON LA INSTALACIÓN DEL TRANSDUCTOR PR4501 SE HABILITA LA INDICACIÓN DEL SENSOR DE NIVEL. 
ESTA INDICACIÓN SE DA EN METROS Y EN LA PUERTA DEL GABINETE SE DEJÓ UNA GRÁFICA DEL CÁLCULO DE ACPM EN M3 VS LA ALTURA EN METROS. 

3. SE CORRIGE EL CONTROL ELÉCTRICO HABILITANDO ASÍ EL NIVEL DE LAS PLANTAS DIÉSEL. 
A LA PLANTA DIÉSEL NUMERO 2 FUE NECESARIO HACER UNA MODIFICACIÓN EXTRA EN EL CONTROL YA QUE EL SENSOR DE NIVEL PRESENTO UN DAÑO. DE IGUAL MANERA QUEDA FUNCIONANDO EN FORMA AUTOMÁTICA Y TODO ESTO ASENTADO EN PLANOS. 

4. SE ENCUENTRA TÉRMICO AVERIADO. EL CUAL FUE REEMPLAZADO POR PERSONAL ELÉCTRICO. 

QUEDA EN OPERACIÓN EL SISTEMA EN MODO AUTOMÁTICO Y MANUAL, LA OPERACIÓN DE LAS BOMBAS DEPENDERÁ DEL NIVEL DEL TANQUE PRINCIPAL. SI EL NIVEL ESTÁ POR DEBAJO DE 0,9M INHABILITARÁ EL ARRANQUE DE LAS BOMBAS DE LLENADO DE LAS PLANTAS DIÉSEL. 
MODO DE OPERACIÓN EN AUTOMÁTICO. 
CUANDO SE TENGA NIVEL EN EL TANQUE PRINCIPAL Y EL NIVEL DE LA PLANTA DIÉSEL ESTE EN NIVEL BAJO, LA BOMBA INICIARÁ AUTOMÁTICAMENTE. TRABAJARÁ HASTA QUE EL NIVEL LLEGUE A NIVEL ALTO. EN ESE PUNTO PARARÁ HASTA QUE EL NIVEL ESTE NUEVAMENTE EN NIVEL BAJO Y ASÍ SUCESIVAMENTE. 

MODO OPERACIÓN MANUAL.
CUANDO SE TENGA NIVEL EN EL TANQUE PRINCIPAL LAS BOMBAS DE LAS PLANTAS SE PODRÁN ENCENDER Y APAGAR SIEMPRE Y CUANDO EL NIVEL DE LAS PLANTAS NO ESTÉ EN NIVEL ALTO.

CABE RESALTAR QUE YA ESTÁ EN PROCESO DE COMPRA 4 SENSORES DE NIVEL DE LAS PLANTAS DIÉSEL, PARA NORMALIZAR EL SISTEMA.
